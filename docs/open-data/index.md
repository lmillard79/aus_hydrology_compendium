## Planned scope

- Bureau of Meteorology — streamflow data (Water Data Online), daily and sub-daily rainfall
- ARR Datahub — design rainfall, temporal patterns, pre-burst ratios, regional parameters
- Geoscience Australia — national digital elevation, catchment boundaries
- State government data portals — QLD, NSW, VIC, SA data access
- Access tips — navigating BOM Water Data Online, extracting GPKG from ARR Datahub
